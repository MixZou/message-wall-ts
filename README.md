# Demo Intro
> Practice basic Create, Delete, Update and Search function
> 
**Vu3 + Vite + TypeScript + Setup + SFC + Vue-Router + Axios**
- Edit message
- Search message by keywords in content
- Delete message
- Update message
- Create message
- The default number of requested information is 10
- NotFound Page
# Deployment
```bash
# install
pnpm install
# run
pnpm run dev
# build
pnpm run build
```
